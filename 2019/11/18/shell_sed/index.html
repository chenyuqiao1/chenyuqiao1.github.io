Linux文本处理三驾马车之一sed

sed对”行”操作

主要包括三个内容

第一：sed的动作：

a，c，d，i，p，s

第二：sed的参数:

-i 直接编辑文本，后接文本
-e 是参数，表示或者的意思
-r 扩展正则表达式
-n 与动作p做搭配使用

第三：sed的一些坑：

双引号和单引号的坑

动作含义

a ：新增， a 接字串，而这些字串会在新的一行出现(目前的下一行)～

c ：取代， c接字串，这些字串可以取代 n1,n2 之间的行！

d ：删除， d 后面通常不接任何；

i ：插入， i 的后面可以接字串，而这些字串会在新的一行出现(目前的上一行)；

p ：打印，通常 p 会与参数 sed -n 一起运行～

s ：取代， + g：全局替换 + i：忽略字符大小写

sed ‘s/要被取代的字串/新的字串/g’

删除固定行的固定内容

其中 ^. 为删除行首第一个字符

^.. 为删除行首头俩字符，以此类推。

或者用正则
