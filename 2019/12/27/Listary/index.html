黑科技软件之快速搜索工具——Listary

Listary是一款适用于Windows的革命性搜索工具，可以为休闲和高级用户提供快速查找文件和快速启动应用程序的功能！

1.下载

Listary 官网下载：https://www.listary.com/download

2.入门使用

  • 启动应用

您可以随时按Ctrl或者Alt + Q两次以激活Listary以搜索所有应用和文件。

当您的搜索查询不包含空格时，Listary会首先列出应用程序。如果您只想搜索文件，只需在搜索查询中的任意位置添加空格即可。

启动应用

  • Listary的2种模式

您可能已经注意到Listary有两种模式。当您使用find-as-you-type激活它时，它会弹出Explorer的右下角。这是文件管理器模式。使用Ctrl X 2热键激活Listary时，它会在屏幕中间弹出。这是启动器模式。

  • 文件管理器模式：此模式侧重于文件搜索。列表搜索栏显示在当前文件管理器窗口（例如资源管理器）中。文件夹在当前文件管理器窗口中打开。应用启动，网络搜索等功能已停用。
  • 启动器模式：此模式下提供所有功能。列表搜索栏显示在屏幕中间。文件夹在新文件管理器窗口中打开。

基本热键

  • 直接在Explorer中输入：激活Listary进行搜索。
  • 按两次Ctrl键：随时随地手动激活Listary。
  • Esc：隐藏列表搜索栏/退出操作。
  • Down ↓或Ctrl + N：选择下一个项目。
  • Up ↑或Ctrl + P：选择上一个项目。
  • Right →或Ctrl + O：输入所选搜索结果的“操作”菜单。

3.热键

  • 热键激活Listary
  • 按Ctrl twice（推荐，适用于所有Windows版本）
  • 按Ctrl twice（Windows 10）
  • Win + G （Windows 8）
  • Win + S （Windows 7 / XP）

您可以在列表选项 - 热键中更改它。

  • 搜索结果热键

您可以使用这些热键来导航搜索结果。在列表选项 - 热键中更改它们。

  • Down arrow或Ctrl + N：选择下一个项目。
  • Up arrow或Ctrl + P：选择上一个项目。
  • Right arrow或Ctrl + O：输入所选搜索结果的“操作”菜单。
  • Esc：隐藏列表搜索栏/退出操作。
  • 命令热键

以下热键在以下情况下工作：

  1. 可以看到列表搜索栏或菜单。您可以直接在文件对话框中使用这些热键，因为默认情况下可以看到列表搜索栏。
  2. 或者在列表选项 - 当前应用程序的应用程序设置中选中“启用列表热键”设置。默认情况下会对Windows资源管理器进行检查，因此您可以直接在资源管理器中使用这些热键，而无需先激活Listary。
  • Ctrl + G：让文件对话框导航到资源管理器中的当前文件夹。
  • Ctrl + Shift + C：将Explorer中所选文件的路径复制到剪贴板。
  • Ctrl + Shift + R：打开当前文件夹的命令提示符。
  • Ctrl + Shift + X：显示/隐藏文件扩展名。
  • Ctrl + Shift + H：显示/隐藏隐藏文件。
  • Ctrl + Shift + O：让文件对话框在资源管理器中打开所选文件。
  • Ctrl + Shift + E：为当前文件夹打开一个新的资源管理器窗口。

您可以在列表选项 - 菜单 - 命令中更改这些热键（从下拉菜单中选择它）。

  • 动作热键

后选择一个搜索结果中Listary，您可以使用以下热键：

  • Ctrl + Enter：打开此搜索结果的父文件夹。
  • Ctrl + Shift + C：将此搜索结果的路径复制到剪贴板。
  • Ctrl + C：将此搜索结果复制到剪贴板（文件格式）。
  • Alt + S：打开发送到此结果的菜单。
  • Alt + O：打开“ 打开”，显示此结果的菜单。

您可以在列表选项 - 操作中更改它们。
